Exercício 1 - Pesquise sites geradores de cores e paletas em css e descreva as funcionalidades de pelo menos 3 deles.

O primeiro site paletadecores.com, possui algumas funcionalidades:

  • - Exibe alguns conjuntos de cores combinadas de forma harmônica, tanto cores diferentes, quanto monocromáticas;
  • - Disponibiliza o código RGB de cada cor a mostra;
  • - Permite gerar um código CSS com as cores da paleta selecionada;
  • - Possibilita que o usuário gere novas paletas;
  • - Possibilita que o usuário extraia cores de uma imagem local ou através de uma imagem do Instagram, através do link da imagem.

O segundo site mybrandnewlogo.com é bem parecido com o primeiro, ele possui algumas funcionalidades:

  • - Exibe alguns conjuntos de cores combinadas de forma harmônica, tanto cores diferentes, quanto monocromáticas;
  • - Disponibiliza o código RGB de cada cor a mostra;
  • - Permite gerar um código CSS com as cores da paleta selecionada;
  • - Exibe estilos diferentes de paletas, como: Cores vibrantes, Cores vintage, Cores cinza ou Cores suaves.

O terceiro site paletadecolores.online possui uma diferença entre os dois anteriores, como:

  • - Possibilita que o usuário gere gradientes, podendo excluir as cores adicionadas por padrão, e adicionar novas cores;
  • - Possibilita a configuração do gradiente: aumentar ou diminuir o tamanho das cores, alterar a orientação, entre outros;
  • - Também disponibiliza o CSS das configurações do gradiente;
  • - Permite o usuário selecionar uma paleta de cores para uma cor específica.
